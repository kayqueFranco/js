FRAMEWORK

Um Framework é um conjunto de ferramentas, biblioteca e regras 
que facilite o desenvolvimento de sofware, fornecendo uma estrutura
pronta para o progamador. O framework define um "esqueleto" do projeto,
onde você preenche com o seu código, seguindo padões  ja estabelecido.
Isso economiza tempo, evita a repetição de código e garante boas práticas.

Um framework é  como econtrar um hamburguer em uma rede de fest food.
sem um framework, você teria que moer a carne, assar o pão,
prepara os ingredientes e criar própria rceita do zero.

EX: Bootstrap(framework front-end)


Passo 1: Acessae a documentação de framework:
https://getbootstrap.com/docs/5.3/getting-started/introduction/

Passo 2: Criar os links para o CSS e JS, conforme orientações 
da documentação 

Passo 3: Criar inicialmente uma classe
